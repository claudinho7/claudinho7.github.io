Russian Roulette

University Solo Work


  • Introduction to programming. My first contact with it, using Arduino and c++ to create a random picker that is using a seed to make sure it is "as random as possible", to simulate a revolver that shoots (a bang! flag) when the number that contains the bullet is picked, also shown by the lights. I am also removing from the array the numbers that have already been picked and are not the ones with the bullet, to further simulate a real gun.
